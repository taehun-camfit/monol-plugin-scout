# Plugin Scout - 작업 완료 후 추천 체크

작업이 완료되었습니다. 다음 조건을 확인하여 플러그인 추천 여부를 결정하세요:

## 추천 조건

1. **이번 세션에서 아직 추천하지 않음** (중복 추천 방지)
2. **의미 있는 작업 완료** (단순 질문/답변이 아닌 코드 작성, 파일 수정 등)
3. **프로젝트 컨텍스트 존재** (package.json, pyproject.toml 등 감지됨)

## 추천 로직

위 조건이 모두 충족되면:

1. 프로젝트 타입 확인 (이미 알고 있다면 스킵)
2. 현재 설치된 플러그인 확인: `~/.claude/settings.json`의 `enabledPlugins`
3. 프로젝트에 유용할 수 있는 미설치 플러그인 1개만 간단히 제안:

```
💡 이 프로젝트에 [plugin-name] 플러그인이 도움될 수 있어요.
   [한 줄 설명]
   관심있으시면 `/scout`로 더 알아보세요.
```

## 제한 사항

- 세션당 최대 1회만 추천
- 너무 길게 설명하지 않음 (2-3줄 이내)
- 사용자가 바쁘면 추천 생략
- `/scout` 명령어를 직접 실행하지 않음 (제안만)

## 추천 안 하는 경우

- 이미 이 세션에서 추천함
- 단순 질문/답변 세션
- 프로젝트 파일이 없음
- 사용자가 플러그인 관련 작업 중
